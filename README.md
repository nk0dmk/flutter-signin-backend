## Backend server for flutter google sign-in app
#### **with firebase authentication**

## How to use

### install deps
```sh
yarn add
# or 
nmp i
```
### copy **.env.example** to **.env**  and **fill** vars
```sh
cp .env-example .env
```
### start server
```sh
npm start
# or
nodemon start
```

#### used packages
 * body-parser
 * dotenv
 * express
 * jsonwebtoken
 * google-auth-library
